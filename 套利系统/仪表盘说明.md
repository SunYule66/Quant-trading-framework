# 套利系统 · 可视化管理仪表盘

## 布局说明

- **顶部 Tab 导航**：概览、回测、结果、配置（侧栏默认收起，主区全宽）
- **概览**：顶部 4 个 KPI 卡片；下方左右两栏——左「快捷回测」（交易对 + 日期 + 执行），右「最近回测摘要」
- **回测**：左栏「数据配置」（交易所/交易对/周期/日期），右栏「策略参数」（网格 / 手数 / 止盈止损 三子 Tab）；底部「执行马丁双向回测」按钮，配置写入 `results/run_config.json`
- **结果**：四个 Tab——马丁摘要、套利结果、交易记录（流水与平仓）、图表
- **配置**：config.py 只读（套利 / 交易 / 风险 / 路径与日志）折叠展示

## 安装依赖

在「套利系统」目录下执行：

```bash
pip install -r requirements.txt
```

## 启动

在「套利系统」目录下执行：

```bash
streamlit run dashboard.py
```

浏览器会自动打开 `http://localhost:8501`。若未自动打开，请手动访问该地址。

## 目录说明

- 仪表盘会读取 `config.py` 与 `results/` 下的 JSON、PNG 结果文件
- 回测结果写入 `results/`，在「概览」或「结果」页查看
